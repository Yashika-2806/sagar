🚀 Building SentinelAI — An AI-Powered Log Intelligence Platform
Modern systems generate thousands of logs every minute, but identifying the real issue behind failures is still a challenge. I started building SentinelAI to explore how AI can help developers and engineers quickly detect anomalies and understand the root cause of system failures.
SentinelAI is designed to analyze application logs, detect unusual patterns, and provide insights that help developers diagnose problems faster.
🔧 Tech Stack I'm Using
• Python & FastAPI for backend APIs
• Redis Streams for real-time log processing
• MongoDB for storing logs and analysis results
• Machine Learning (TF-IDF + Autoencoder) for anomaly detection
• React for the monitoring dashboard
• Docker for containerized deployment
The goal is to build a system that can move from raw logs → anomaly detection → root cause insights → remediation guidance.
Excited to keep improving it and learning along the way.
More updates soon as the project evolves.
This project is being developed under the guidance of my mentor, Preshit Desai
